# quizshow

a collaborative realtime quiz show

# Requires

* [Node.js](http://nodejs.org)
* [npm](http://npmjs.org)
* [mongodb](http://mongodb.org)

# Install

    $ git clone https://github.com/ecarter/quizshow.git
    $ cd quizshow
    $ npm install

# Usage

Start the show!

    $ npm start
    $ open http://localhost:3000
